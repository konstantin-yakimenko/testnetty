## Сущности
###### Country - Страна
* Long id - ключ
* String title - наименование

###### City - Город
* Long id - ключ
* String title - наименование

## API

* GET /city/:id - поиск города 200
* POST /city - создать 201
* PUT /city - изменить 200


* GET /country/:id - поиск страны 200
* POST /country - создать 201
* PUT /country - изменить 200
